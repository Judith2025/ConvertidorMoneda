## :hammer: Descripcion de proyecto. 

El proyecto consta de un menu el cual desplega la opcion de moneda a convertir.
  
Los pasos para completar del desafío por AluraLatam se detallarán a continuación:

-Configuración del Ambiente Java.

-Creación del Proyecto.

-Consumo de la API.

-Análisis de la Respuesta JSON.

-Filtro de Monedas.

-Exibición de Resultados a los usuarios.
