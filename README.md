<h2 align="center">Challenge ONE | AluraLatam Oracle.</h2> 

Implementación de un conversor de divisas en Java que utiliza la conexión a una API externa, para obtener tasas de cambio en tiempo real. El usuario interactúa con la aplicación mediante un menú para especificar la conversion que desea.
 
## :hammer: Descripcion de proyecto. 

El desarrolló del programa facilita la conversión de divisas. La interfaz de usuario opera mediante un menú, donde el usuario especifica la conversión y la cantidad deseada. La funcionalidad clave reside en su integración con la ExchangeRate-API, lo que permite obtener las tasas de cambio más recientes y aplicar el cálculo con datos en tiempo real. 

El proyecto conta de:

-Configuración del Ambiente Java.

-Consumo de la API.

-Análisis de la Respuesta JSON.

-Filtro de Monedas.

-Exibición de Resultados a los usuarios.



:currency_exchange: **Monedas Soportadas.**

- Dolar
- Peso Mexicano
- Real Brasileno
- Euros
